About SocialNotes

A modern platform for sharing thoughts and ideas

What is SocialNotes?

SocialNotes is a modern, sleek note-sharing platform developed by Jozef for his Internet & Web Programming class at Fordham University (CISC3300) taught by Professor Nate Hishon.

Our Features

  • User Accounts

    Create your personal account to manage and own your notes

  • Intuitive Note Creation

    Create and edit notes with a clean, distraction-free interface

  • Mood Tags

    Mark notes with different moods using expressive emojis

  • Privacy Control

    Only you can edit or delete your own notes

  • Dark Mode Design

    Easy on the eyes with a modern dark theme

Technology

SocialNotes is built using modern web technologies including HTML5, CSS3, JavaScript, PHP, and MySQL. The platform follows an MVC architecture with a RESTful API for seamless interactions between the frontend and backend.
